jQuery.Tacky

Take your nav and stick it up your page.

  • Create
  • Customize
  • Initialize
  • Go Crazy

1. Create the Nav and Sections

Tacky automatically detects the links in your nav which keeps your code clean. Stay semantic or customize it to your exact needs.

2. Customize the CSS

Style the navigation however you want, use media queries to mobilize it.

3. Initialize jQuery.Tacky

Tacky comes with a lot of useful options. The parentSelector for example will look for the closest parent element and apply the active class to it.

4. Go Crazy

Lacking a feature you want? Let us know or go fork yourself. jQuery.Tacky is MIT Licensed.
